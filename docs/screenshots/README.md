# Screenshots

This folder contains approved UI screenshots for documentation and reference.

## Approved UI (v1)

The current locked UI direction features:
- Vibrant, modern aesthetic
- Layered background with depth
- Glassmorphism central card
- Teal/aqua/cyan accents
- Deep blue base

## Adding Screenshots

When adding screenshots:
1. Use English language setting
2. Set status to appropriate state (Idle/Connecting/Secure)
3. Ensure demo mode notice is visible
4. No error messages in view
5. Clean URL in input field

## Naming Convention

- `ui-idle.png` — Idle state
- `ui-connecting.png` — Connecting state
- `ui-secure.png` — Secure/Connected state
- `privacy-drawer.png` — Privacy drawer open
