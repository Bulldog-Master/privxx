# Privxx Connection State Machine

## States

### Idle
- Default state
- Button enabled
- Status: Idle

### Connecting…
- Triggered on Connect click
- Button disabled
- Subtle background motion
- Status: Connecting…

### Secure
- Triggered when connection completes
- Button shows Connected
- Status: Secure
- Background motion stops

## Rules
- No fake progress bars
- No percentages
- No misleading encryption claims

## Demo Mode
Until cMixx is fully integrated, the UI must indicate:
"Demo mode — routing simulated"
