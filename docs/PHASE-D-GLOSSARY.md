# Phase D Glossary (Plain English)

## cMixx
A privacy network that prevents metadata correlation by mixing traffic.

## xxDK
The developer kit used to integrate cMixx into an application.

## Control Channel
A private message path used to coordinate actions (connect/status), not full browsing traffic yet.

## Event-Driven Secure State
Secure status changes only when a real cMixx message is confirmed (not a UI timer).

## Listener / Server
A small service that receives Privxx messages and returns responses.

## Proof of Integration
A minimal demo that proves messages go through cMixx and back successfully.
